def goes_after(word: str, first: str, second: str) -> bool:

    return -1 < word.find(first + second)

Aber in dieser Mission können dieselben Zeichen mehrmals in word vorkommen. Sie müssen also überprüfen, ob die Teilzeichenkette nicht nur in word vorkommt, sondern auch das erste Auftreten der Zeichen einschließt.

Ok, was soll ich als nächstes tun?

Die vorhandene Lösung prüft bereits das Vorhandensein der Teilzeichenkette in word. Nehmen wir an, sie ist vorhanden. Die Teilzeichenkette beginnt mit dem Zeichen first und es muss das erste Vorkommen des ersten Zeichens sein. Daher muss das Ergebnis der Methode find() für das Zeichen first dasselbe sein wie für die Teilzeichenkette! Es ist möglich, den Vergleich zu ergänzen. 

def goes_after(word: str, first: str, second: str) -> bool:

    return -1 < word.find(first + second) == word.find(first)

Ich brauche einen Hinweis auf meinen nächsten Schritt.

Wenn Sie mit der gleichen Logik vorgehen, muss der Index der Teilzeichenkette (und das erste Auftreten des Zeichens first ) kleiner sein als der Index des ersten Auftretens des Zeichens second.

Wie sollte also meine endgültige Lösung aussehen?

Hier ist sie! 

def goes_after(word: str, first: str, second: str) -> bool:

    return -1 < word.find(first + second) == word.find(first) < word.find(second)

Da die Zeichen von second direkt nach first stehen sollten, können Sie auch die folgende Variante schreiben: 

def goes_after(word: str, first: str, second: str) -> bool:

    return -1 < word.find(first + second) == word.find(first) == word.find(second) - 1

Sicher, greifen Sie zu!

Da die Methode index() bei Nichtvorhandensein einen Fehler auslöst, benötigen Sie die Anweisung try-except. Das erste Vorkommen des second Zeichens sollte das nächste nach dem first sein. 

def goes_after(word: str, first: str, second: str) -> bool:

    try:
        return word.index(second) - word.index(first) == 1
    except ValueError:
        return False

Diese Variante ist sogar kürzer als unsere und logisch vollständig! 

def goes_after(word: str, first: str, second: str) -> bool:

    return -1 < word.find(first) == word.find(second) - 1
